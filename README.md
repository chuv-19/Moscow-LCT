# 🌲 Система обнаружения деревьев

Система на базе искусственного интеллекта для обнаружения деревьев и выявления их дефектов с использованием модели **YOLOv11**.

---

# 🚀 Быстрый старт (для новых пользователей)

## Установка одной командой

**Mac/Linux:**

```bash
chmod +x setup.sh
./setup.sh
```

**Windows:**

```cmd
setup.bat
```

Установка займет всего **5–10 минут** и автоматически настроит все необходимые зависимости.

## Запуск приложения

**Mac/Linux:**

```bash
./start_mobile_app.sh
```

**Windows:**

```cmd
start_mobile_app.bat
```

После запуска приложение будет доступно по адресу:

**http://localhost:8501**

---

# 📱 Установка приложения на телефон (PWA)

## Шаг 1. Запуск с доступом по сети

```bash
./start_app_network.sh   # Mac/Linux
start_app_network.bat    # Windows
```

## Шаг 2. Подключение с телефона

1. Узнайте IP-адрес компьютера:

   * **Mac:** `ifconfig | grep "inet "`
   * **Windows:** `ipconfig`
2. На телефоне откройте браузер и перейдите по адресу:

   ```
   http://ВАШ_IP:8501
   ```
3. В меню браузера выберите **«Установить приложение»**.
4. После установки приложение появится на главном экране телефона.

Подробная инструкция находится в файле **PWA_GUIDE.md**.

# ✨ Основные возможности

## Обнаружение объектов

### 🌳 Обнаружение деревьев

Поддерживаются следующие классы:

* Кустарник (Bush)
* Дуб (Oak)

### 🔍 Обнаружение дефектов

Поддерживается **12 типов дефектов**, включая:

* Трещина (Crack)
* Засохший куст (Dead Bush)
* Засохшее дерево (Dead Tree)
* Усыхание кроны (Dry Crown)
* Наклонённое дерево (Leaned Tree)
* Помеченное дерево (Marked Tree)
* Гниль (Rot)
* Повреждение ствола (Stem Damage)
* Гниль ствола (Stem Rot)
* Дупло (Tree Hole)
* и другие виды дефектов.

# 🐞 Решение возможных проблем

## Ошибки установки

### «Python not found»

Установите Python версии **3.8** или выше с официального сайта python.org.

### «Permission denied»

```bash
chmod +x setup.sh
./setup.sh
```

---

## Ошибки во время работы

### «Model not found»

Сначала обучите модели:

```bash
./train_trees.bat
./train_defects.bat
```

### «Port already in use»

Освободите порт 8501:

```bash
lsof -ti:8501 | xargs kill -9
```

### Телефон не подключается

Проверьте:

* что компьютер и телефон находятся в одной сети Wi-Fi;
* правильность IP-адреса;
* настройки брандмауэра;
* использование протокола **http://**, а не **https://**.

Дополнительные рекомендации содержатся в файле **QUICK_START_NEW_USER.md**.
